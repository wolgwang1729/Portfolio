# Portfolio

## Description

This is a personal portfolio website for wolgwang, a CS student at DTU. The site showcases his background in Systems Engineering and Computer Vision, highlighting his education, skills, projects, internships, achievements, and contact information. It serves as a professional online presence to demonstrate expertise in AI/ML, web development, and open-source contributions.

## Features

- **Hero Section**: Animated introduction with name, tagline, and call-to-action.
- **About Section**: Personal story, education details, and philosophy.
- **Tech Stack Section**: Categorized skills in Languages, Tools, Frameworks, Libraries, and AI/ML.
- **Experience Section**: Timeline of internships and project details.
- **Projects Section**: Showcase of featured projects with descriptions, tech tags, and links.
- **Profiles Section**: Links to professional and competitive programming profiles.
- **Contact Section**: Direct links to email and social media.

## Tech Stack

- Framework: Next.js 16.1.6
- Language: TypeScript 5.9.3
- Styling: Tailwind CSS v4.1.18
- Animations: Framer Motion
- Icons: Lucide React and React Icons
- Fonts: Geist Sans and Geist Mono
- Other: clsx, tailwind-merge, PostCSS, ESLint

## Prerequisites

- Node.js (compatible with Next.js 16.1.6)

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd portfolio
   ```

2. Install dependencies:
   ```
   npm install
   ```

## Usage

- Run development server:
  ```
  npm run dev
  ```
  Opens at http://localhost:3000

- Build for production:
  ```
  npm run build
  ```

- Start production server:
  ```
  npm run start
  ```

- Lint code:
  ```
  npm run lint
  ```

## Deployment

Deploy to Vercel by connecting the GitHub repository to Vercel for automatic builds.
